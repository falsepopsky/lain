# @untidy/animeflv

## :information_source: About

## :sparkles: Features

- Built with TypeScript for full typing support
- ESM only

## :package: Install

With npm:

```
npm install @untidy/animeflv
```

With yarn:

```
yarn add @untidy/animeflv
```

With pnpm:

```
pnpm add @untidy/animeflv
```
